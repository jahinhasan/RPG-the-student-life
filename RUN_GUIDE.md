# How to Run Your RPG App 🚀

Follow these steps to run the app on your Android emulator at any time:

### 🎮 1. Start the Emulator
Run this command in your terminal to launch the emulator:
```bash
flutter emulators --launch Medium_Phone_API_36.1
```
*Wait about 1-2 minutes for the "Android" screen to finish loading.*

### 🚀 2. Run the App
Once the emulator is showing its home screen, run:
```bash
flutter run
```

### 💡 Quick Tips
- **Hot Reload**: Press `r` in the terminal after making changes to see them instantly!
- **List Devices**: If you're not sure if the emulator is ready, run `flutter devices`.
- **Firebase**: Since we've integrated Firebase, the app might take a few seconds more to initialize on the first run.

---
*Note: If you ever see "device is still booting", just wait 30 seconds and try `flutter run` again.*
